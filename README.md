# MULTI-STORE-SALES-DASHBOARD
# Multi-Store Sales Dashboard Project  ## Project Overview This project analyses multi-store sales data to generate business insights and visual dashboards.  It includes:  - Store-wise total sales and revenue trends - Product category-wise sales performance - Top selling products across stores - Month-wise and quarter-wise sales
